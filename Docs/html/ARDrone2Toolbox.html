AR Drone 2.0 Toolbox

The AR Drone 2.0 Toolbox provides hardware support for the Parrot AR Drone 2.0, a commercially available quad-copter. The provided Simulink library blocks can read the multitude of sensors as well as control the different actuators on the drone.

Contents:

The Examples section provides examples of how to run pre-made demo's or custom models on the AR Drone 2.0.

The Documentation section contains information about the drone hardware and the software implementation.
